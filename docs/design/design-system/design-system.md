# Design System - COCO

## Principios

- Claridad: jerarquia visual legible.
- Consistencia: mismos tokens para componentes equivalentes.
- Accesibilidad: contraste y estados claros.

## Paleta base

- Primario: `#8B4513`
- Fondo claro: `#F7F4EF`
- Fondo oscuro: `#1D1F20`
- Texto claro: `#1F2937`
- Texto oscuro: `#FFFFFF`
- Exito: `#16A34A`
- Warning: `#F59E0B`
- Error: `#DC2626`

## Tipografia

- Fuente principal: `Inter`
- Escala sugerida:
  - `h1`: 32px / semibold
  - `h2`: 28px / semibold
  - `h3`: 24px / semibold
  - `body`: 16px / regular
  - `label`: 14px / semibold

## Componentes base

- Botones: primario, secundario, deshabilitado.
- Inputs: borde, placeholder, error/help text.
- Modales: jerarquia clara de acciones.
- Cards: contenedores de dashboard/listados.

## Estados UX obligatorios

- `loading`
- `empty`
- `error`
- `success`
