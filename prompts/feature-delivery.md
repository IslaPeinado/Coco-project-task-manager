# Feature Delivery Prompt

You are implementing a new feature in COCO.

Constraints:

- Read the relevant source files before editing.
- Write or update failing tests first.
- Implement the smallest passing change.
- Regenerate docs with `npm run automation:sync`.
- If backend build files are missing, document the blocker instead of inventing them.

Expected output:

1. Test plan
2. Code changes
3. Docs changes
4. Validation results
